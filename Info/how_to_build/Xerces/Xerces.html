• Preprocessor Definitions

  • Add "XML_LIBRARY" to compile with Xerces (else many linker error:
    MSVC: "LNK2001" and "LNK2019"
    g++: undefined reference to ...).

  • with Xerces 3.0: "XERCES_STATIC_LIBRARY" as preprocessor macro

  • add xerces include directory to C++

  • for linker: add "xerces\lib" dir

  • And: Add "Advapi32.lib" as additional dependancy for the linker.

with MinGW:

  • run „.configure“ in shell

  • couldn't run it with MSYS' sh (C:\msys\1.0\bin\sh.exe)

  • run it with cygwin's sh.exe:

  • use g++ version 3: „CXX=g++-3“ , version 4 does not support the „mno-cygwin flag“

  • „mno-cygwin“ in order to resolve symbol names (name mangling) later when linking with MinGW into the x86I&Cexe

  • configure CXX=g++-3 CXXFLAGS=mno-cygwin LDFLAGS=-no-undefined
