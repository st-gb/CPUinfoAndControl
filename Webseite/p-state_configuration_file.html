for the service (also possidsdble for the GUI) to select the voltage for the on-demand performance switching (Dynamic Voltage and Frequency Scaling)
  • create a configuration file in XML format if none exists
    • the path to it is "configuration/CPU Vendor/CPU family/CPU model/CPU stepping/CPU name.xml" releative to the working dir of the executable.
      e.g. "C:\X86Info_and_Control\configuration\GenuineIntel\6\13\6\Intel(R) Pentium(R) M processor 1.80GHz.xml" for a Pentium M 745
    • You can use Notepad or special XML editors to create and edit the file.
  • The structure of the configuration file should be like:

    <topmost_XML_Element>
        <    freq_and_voltage frequency_in_MHz="FreqInMHz" min_voltage_in_Volt="MinVoltageInVolt" voltage_in_Volt="WantedVoltageInVolt" max_voltage_in_Volt="MaxVoltageInVolt"/></freq_and_voltage>
    </    topmost_XML_Element>

  • explanations for the XML attributes/ elements:
    topmost_XML_Element:
    an XML document needs a superordinate XML element. You can name it like you want but it is a good idea to give it the same name as the CPU name (=the XML file name without the file extension)
    freq_and_voltage:
    You need to test at least 2 frequencies:
    • the maximum frequency
      and 
    • a low /the lowest frequency.
    The voltages for the intermediate frequencies are interpolated if not specified.
    MaxVoltageInVolt:
    should be the default voltage for the freq.
    WantedVoltageInVolt
    the voltage to set
    MinVoltageInVolt
    suggestion: set this to the lowest stable voltage
