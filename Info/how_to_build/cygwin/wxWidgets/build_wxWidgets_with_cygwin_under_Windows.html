build wxWidgets with cygwin under Windows

  1. running "make makefile.gcc"/ "gnuwin32-make makefile.gcc" didn't work

  2. download wxWidgets for Windows, run setup

  3. Ensure that there are NO spaces in the dir pathes!

  4. inside the Windows-Shell ("cmd.exe"; prompt), start the Unix-Shell "sh" (>>cygwin dir<<\bin\sh.exe)

  5. run the script ">>wxwidgets-dir<</configure ..."

  6. executing "make" failed at ">>wxwidgets-dir<</utils/wxrc" if I recall correctly
