  • download and install via "Devel->libxerces" via Cygwin installer

    • only release build of a project working when linking with Cygwin's xerces

      • debug build:
             -keine xerces Include-Verzeichniss angeben, sonst Linker-Fehler
               "undefined reference to `xercesc_3_0::XMLString::transcode(unsigned
                short const*, xercesc_3_0::MemoryManager*)'"
             "--enable-auto-import" als Linker-Option damit
               -"xercesc_3_0::XMLPlatformUtils::fgMemoryManager"
               -xercesc_3_0::XMLUni::fgSAX2CoreValidation
               -xercesc_3_0::XMLUni::fgSAX2CoreNameSpaces
               gelinkt werden

  • build xerces libraries:
       -via make:
        -1. open cmd.exe
        -2. run "bash" within cmd shell
        -3. use "./configure LDFLAGS=-no-undefined  --disable-network"
          "--disable-network" because <sys/socket.h> couldn't be found
